

Turbine Tutorials
-----------------

This section describes the wind turbine module of FUSED-Wind through a series of tutorials.
The module contains several sub-modules related to e.g. geometry, aerodynamic, aeroelastic, and structural models.
In the following tutorials each of these modules is described in more detail, accompanied with examples of how to use them.
